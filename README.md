# xpertik-odontograma

Custom Django Field + Widget for FDI/ISO-3950 odontograms.

Capture per-tooth, per-face dental findings from Django admin or any ModelForm — no AJAX, no extra endpoints.

Full documentation (install, quickstart, extending states, readonly widget, logger namespaces, CI badge) will be published in Phase 5 of the v0.1.0 scaffolding change. See `CHANGELOG.md` for release status.
